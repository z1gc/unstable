# Unstable Gentoo Overlay

For personal flavor, without much tested, might break your system :/

Contains only live packages, and many of them are hacked with some features that
only suit my personal demands, that's why it's unstable.

Better plays with [smart-live-rebuild](https://github.com/projg2/smart-live-rebuild/)
if you really want to use it.

# .*

Directories beside from Gentoo overlay:

* `pygoscelis`: System configurations
* `aptenodytes`: Dotfiles of mine

They are managed by [comtrya](https://github.com/comtrya/comtrya).
